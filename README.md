## 📋 Descripción

Aplicación web educativa desarrollada con Flask para el aprendizaje interactivo de **adjetivos derivados en español**. La plataforma ofrece 100 ejercicios, flashcards interactivas, práctica personalizada y seguimiento de progreso.

## ✨ Características Principales

### 🎯 **Aprendizaje Estructurado**
- **100 adjetivos derivados** con explicaciones detalladas
- **Clasificación semántica**: Calificativos vs. Relacionales
- **3 niveles de dificultad**: Básico, Intermedio, Avanzado
- **21 sufijos diferentes** con sus significados específicos

### 🃏 **Sistema de Flashcards**
- Tarjetas interactivas con animación 3D
- 3 modos de estudio: Aprender, Repasar, Evaluar
- Sistema de memoria con seguimiento de recordadas/olvidadas
- Estadísticas de progreso en tiempo real
- Atajo de teclado (espacio para voltear)

### ✍️ **Práctica Interactiva**
- Ejercicios de formación de adjetivos
- Retroalimentación inmediata
- Sistema de puntuación
- Configuración de dificultad y número de preguntas

### 📊 **Seguimiento de Progreso**
- Historial de resultados
- Estadísticas de rendimiento
- Métricas por nivel y tipo de adjetivo
